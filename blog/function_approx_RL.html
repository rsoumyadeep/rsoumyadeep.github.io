Why Do We Need Function Approximation?

Real-world RL environments have enormous or continuous state spaces.

Curse of Dimensionality

Tabular methods explode in memory. A 10D state space with 10 points per dimension has 10¹⁰ states.

Generalization

Similar states should have similar values. Function approximators learn smooth patterns across the state space.

Scalability

Neural networks can handle millions of parameters and learn directly from raw pixels (Atari, robotics).
