How Attention Works

The model computes Query (Q), Key (K), and Value (V) vectors for each token.

It then calculates attention scores as:

softmax( QKT / √dₖ )

These scores determine how much each token should "attend" to others. The final output is a weighted sum of Value vectors.

This mechanism allows the model to focus on relevant parts of the input dynamically.
